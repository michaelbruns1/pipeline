Fixing Unstyled Flash

In this topic, we discuss a common issue of an unstyled flash on an HTML page that references a CSS file, and its solution.

Problem

I see a flash of unstyled content on my website before the styles kick-in. How do I fix it?

Cause

When a page load includes a network request to retrieve a CSS file, an unstyled flash may occur during the load. This is due to the time it takes for the network request to retrieve the CSS file.

Embedding CSS Styles Directly in HTML

Embed the CSS styles directly within the HTML element associated with it to resolve this issue.

Embed the CSS styles directly within the HTML element associated with it to resolve this issue.

Follow these steps:

  1. Create an HTML page that references a CSS file where unstyled flash occurs upon reloading due to the network request to retrieve the CSS file.

  2. Instead of referencing, embed the CSS directly in the HTML element associated with the CSS by using the element's #ID tag.

  3. Reload the webpage. The content should now load without an unstyled flash as no network request is needed to retrieve the CSS.
